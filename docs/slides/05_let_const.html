Let Vs const

Les déclaration let définissent des variables dont la portée est limitée au bloc courant ({}}) 
        
        const key = 'abcd12134';
        let points = '50';
        let winner = false;

        points=60;
        key = 'ababababa'
        const person = {
                name :'gilles',
                age:'44'
        }
Ne pas oublier de parler de Object.freeze (key) et de changer l'objet mais l'intérieur
